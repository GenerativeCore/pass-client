# How to install

- [ ] Install NodeJs 16+
- [ ] Run in terminal

```
npm ci
```

- [ ] Put credentials to `~/consts.ts`

```ts
export const AUTH = {
  username: '?',
  password: '?',
};
```

- [ ] Create folder `~/images`

# How to use

## Text to image

- [ ] Run in terminal

```
npm run txt2img
```

- [ ] Check `images` folder
- [ ] Check file `txt2img.ts`
